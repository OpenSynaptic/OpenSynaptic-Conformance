# Documentation

This directory contains the normative and operational documentation for OpenSynaptic conformance work.

Current focus areas:

- `whitepaper/` for architecture, protocol, and alignment material
- `certification/` for level definitions, evaluation flow, and reporting requirements

Additional sections such as protocol reference and operator guides can be added as the repository matures.
