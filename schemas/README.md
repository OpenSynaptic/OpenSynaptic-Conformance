# Schemas

Schemas define the machine-readable contracts used by this repository.

Planned schema families include:

- profile definitions
- vector file formats
- dataset manifests
- conformance report output
- compatibility matrix summaries

Schema changes should be versioned whenever they affect public tooling or stored artifacts.

Current schemas:

- [Profile schema](profile.schema.json)
- [Vector-set schema](vector-set.schema.json)
- [Report schema](report.schema.json)
- [Dataset-manifest schema](dataset-manifest.schema.json)
- [Adapter-manifest schema](adapter-manifest.schema.json)
